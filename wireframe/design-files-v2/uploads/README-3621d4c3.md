# Mini-batches for Batch 6 — Settings + Profile

22 pakker delt i 4 mini-batches for parallell-bulk-generering i claude.ai/design.

| Mini-batch | Pakker | Tema | Antall enheter |
|---|---|---|---|
| 6-A | 1–5 | CoachHQ Settings (profil, bruker, sikkerhet, API, tilgjengelighet) | 5 |
| 6-B | 6–10 | PlayerHQ Settings + shared layout | 5 |
| 6-C | 11–15 | Tverrgående settings + CoachHQ team | 5 |
| 6-D | 16–22 | CoachHQ verktøy + 3 modaler | 7 |

## Slik bruker du

For hver mini-batch:

1. Last opp engang per ny chat-sesjon:
   - `branding-style-guide.html`
   - `design-system-v2.md`
   - Alle 20 .woff2-filer fra `fonts/`

2. Last opp `6-X.md` (mini-batch-spec)

3. Last opp HTML-vedlegg listet i `6-X-vedlegg.txt`

4. Kopier hele `6-X-prompt.md` som første melding

5. La Claude Design generere alle skjermer i ett løp

6. Lim design-links tilbake til Claude Code når ferdig

## Rekkefølge-anbefaling

Kjør 6-A først (etablerer CoachHQ-settings-mønster). Deretter 6-B (PlayerHQ-speiling). 6-C kan kjøres parallelt med 6-D hvis du har to chat-sesjoner åpne.

Estimert tid per mini-batch: 60–90 minutter aktiv iterasjon, mer hvis du krever flere revisjons-runder.
