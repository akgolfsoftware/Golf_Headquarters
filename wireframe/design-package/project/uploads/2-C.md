## Felles arketype-B spec (gjelder alle skjermer her)

# Batch 2 -- List + filter (Arketype B)

**Antall pakker:** 22 (18 skjermer + 4 tilhoerende modaler)
**Status:** Klar for claude.ai/design
**Estimert tid:** 4-6 timer

## Hvorfor denne batchen naa

Arketype B (List + filter) er det mest brukte moensteret i hele plattformen -- dominerer baade CoachHQ og PlayerHQ. Naar dette moensteret er stabilt, kan resterende lister i batch 6+ bruke samme grunnmal. Pilot-batch 1 (Dashboard) etablerte estetikken; denne batchen etablerer **interaksjonsmoenstre for tabell-data**.

## Arketype B -- felles spec (gjelder alle 18 skjermer)

Disse moenstrene skal vaere konsistente paa tvers av alle list-skjermer. Vis variasjon i innhold og kolonner, ikke i layout.

### Layout

```
+---------------------------------------------+
|  Sidebar  |  Hero (italic editorial title)  |
|           +---------------------------------+
|           |  KPI-strip (4 kort, valgfritt)  |
|           +---------------------------------+
|           |  Filter-bar:                    |
|           |  [Soek] [Chip 1] [Chip 2] [Sort] |
|           |                          [+ Ny] |
|           +---------------------------------+
|           |  Tabell / kort-grid             |
|           |  +--+---------+-----+----+---+  |
|           |  |  | navn    | ... |... | -> |  |
|           |  +--+---------+-----+----+---+  |
|           +---------------------------------+
|           |  Pagination eller "Last mer v"  |
|           +---------------------------------+
```

### Kolonner som ALLTID maa designes

| Element | States aa designe |
|---|---|
| Soek-felt | default, focus, with-text, clear-button, no-results |
| Filter-chip (multi-select) | default, hover, selected, disabled, count-badge |
| Sort-dropdown | default, open, item-hover, item-selected (med pil) |
| Sort-icon paa kolonne-header | default, hover, sort-asc, sort-desc |
| Rad | default, hover (subtil bg-shift), selected, klikk -> detail |
| Status-pill (per status) | unike farger per status -- bruk pyramide-paletten |
| "..."-aksjons-meny per rad | default, hover, popover-open |
| Bulk-checkbox | uvalgt, valgt, partial (hvis grupperinger) |
| "+ Ny X" CTA (primary) | default, hover, active, focus, disabled, loading |
| Pagination | default per side, hover, active page, disabled (forrige/neste) |

### Empty / loading / error / count-states

- **Empty (ingen data ennaa):** Stort sentrert omraade med ikon + "Ingen {ressurs} ennaa. Lag din foerste ->" CTA
- **Empty (filter gir 0):** "Ingen treff for filteret. Tilbakestill ->"
- **Loading:** 5 graa skeleton-rader
- **Error:** Per-tabell-error med retry
- **Count:** "Viser 12 av 38" eller "12 spillere" etter filter

### Mobil-versjon

- Filter-bar collapser til en "Filter (3) v"-knapp som aapner bottom-sheet
- Tabell konverterer til kort (en kort per rad, alle kolonner stables vertikalt)
- Bulk-actions skjules; "..."-meny per rad bevares

### Responsive breakpoints

- Desktop: >=1024px -- full tabell, alle filtre synlige
- Tablet: 768-1023px -- tabell med kondenserte kolonner, filtre delvis synlige
- Mobil: <=640px -- kort-layout

## Spec -- hva skjermen er for

Baner er Markus' bibliotek over golfbaner -- baade de han har spilt (8 stk i 2026) og anbefalte han boer proeve. Skjermen kombinerer kart-utforskning med tabbede lister. Han bruker den naar han planlegger en runde og vil sjekke distanse, beste score, eller ny bane i naerheten. GolfBox-import lar ham trekke historikk fra GolfBox-konto.

## Layout -- UNIKT for denne skjermen

Bruker arketype-B-felles-spec fra `README.md`. I tillegg:

- **Hero italic Instrument Serif 36px:** *"8 baner spilt, Markus."* Sub: "Hjemmebane: GFGK - Beste: Borre 74 (-1)"
- **Kart oeverst (300px hoey, full bredde):**
  - Lyst kart-tile med 8 spilte pins (accent) + 6 anbefalte pins (primary outline)
  - Markus' lokasjon-prikk (Fredrikstad) i sentrum med lucide `MapPin`

- **Tabs under kart (3):**
  - **Spilte** (8) -- default
  - **Anbefalt** (6) -- Pro-AI-personalisering

- **Bane-card-grid 3xN:**
  - Card 280x220px
  - Topp-thumbnail: bane-foto (placeholder groenn-gradient med flagg-ikon)
  - Tittel (Inter Tight 16px semibold): "Borre Golfklubb"

  - Meta-rad-2: lucide `MapPin` + "142 km fra hjem"
