## Designsystem

Bruk `branding-style-guide.html` + `design-system-v2.md` + `web/_shared.css`.

## Spec — hva siden er for

404-side for ikke-funne URLer. Tonen skal vaere lett-humoristisk men nyttig
(avoid "oops!"-cliche). Skal lede tilbake til de viktigste sidene.

## Layout — UNIKT

### 1. Top-nav (samme som alle andre web-sider)

### 2. Sentrert hero (lys, full viewport-hoyde minus nav + footer)

Sentrert content max 640px:
- Stor `404` (Inter Tight 240px italic, lime fade fra accent til subtle gra)
- H1 (Inter Tight 56px italic): `Hmm. Den siden er *out of bounds*.`
- Sub: `Ballen havnet ikke der vi trodde. Vi tror dette er det du leter etter:`
- 4 quick-links som cards i grid-2:
  - Forsiden -> /
  - Tjenester -> /tjenester
  - Coaches -> /coaches
  - Kontakt -> /kontakt
- Sekundaer: `Sok i stedet ↓` (toggle som apner sok-felt)

### 3. Sok-felt (lys, valgfri, vises ved klikk)

Stort sok-felt: `Hva leter du etter?`
Submit -> /sok?q={q} (kan eksistere senere)

### 4. Mørkt CTA-band (kort)

`Trenger du hjelp?` -> `Kontakt oss →` (lime) + `+47 482 35 700` (outline)

### 5. Footer
