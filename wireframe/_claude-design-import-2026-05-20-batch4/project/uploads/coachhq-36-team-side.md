# CoachHQ — Team-side

**Rute:** `/admin/team`.

## Kontekst
AK Golf Group har flere team-medlemmer: Anders (hovedcoach), assistenter, fysisk-trener, sportspsykolog, admin. Side for å se og styre alle.

## Formål
- Liste alle teammedlemmer
- Roller og tilganger
- Invitere nye

## Layout

**Header:**
- "Team · 6 medlemmer" Inter Tight 700 32px
- "Hovedcoach Anders · 3 coacher · 2 admin" mono
- "Inviter medlem" forest fill

**Team-grid (3 kolonner):**
Hvert medlems-kort:
- Avatar 80px
- Navn Inter Tight 600 18px
- Rolle-pill (HOVEDCOACH lime / COACH / ADMIN / FYSISK)
- Antall spillere tildelt: "12 spillere"
- Sist aktiv: "I dag" mono muted
- Hover: "Rediger" / "Tilgang" / "Fjern fra team"

Anders' kort har lime-border og "Eier"-badge.

**Tomtilstand for invitasjoner:**
"Ingen ventende invitasjoner"

**Pending-seksjon:**
Hvis invitasjoner ute: "2 ventende invitasjoner" med navn + sendt-dato + "Send på nytt" / "Kanseller".

## Branding
Cream bg, hvite team-kort, lime hovedcoach-border, forest CTA.
