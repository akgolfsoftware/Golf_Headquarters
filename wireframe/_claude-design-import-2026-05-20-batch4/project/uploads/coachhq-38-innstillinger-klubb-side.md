# CoachHQ — Innstillinger klubb (side)

**Rute:** `/admin/settings`.

## Kontekst
Sentral innstillinger for AK Golf Academy som organisasjon: branding, kontaktinfo, jurisk, betalingsmetoder.

## Formål
- Endre klubb-info som ses i e-poster, fakturaer, PlayerHQ-hero
- Last opp logo + branding
- Sette default-betalingsmetode

## Layout

**Header:**
- "Innstillinger" Inter Tight 700 32px
- "AK Golf Academy · Org.nr 998 877 665" mono

**Tab-sidebar venstre:**
- Klubb (default)
- API & integrasjoner
- Kalender
- Sikkerhet
- Tilgang

### Klubb-tab
**Logo-card:**
- Stort logo preview 200×200
- "Last opp ny logo" outline
- Spesifikasjon "PNG/SVG, min 512×512"

**Klubb-info-felt:**
- Klubb-navn
- Org.nr
- Adresse (street, postnr, by, land)
- Telefon + e-post
- Nettside (URL)

**Branding-farger:**
- Primary, accent (med color picker — default lime/forest, men kan tilpasses for white-label-kunder)

**Default-betalingsmetode:**
Stripe-konto-info + "Endre i Stripe Dashboard"-link

**Footer (sticky):**
"Lagre endringer" forest fill

## Branding
Cream bg, hvit panel, forest CTA, mono i org.nr.
