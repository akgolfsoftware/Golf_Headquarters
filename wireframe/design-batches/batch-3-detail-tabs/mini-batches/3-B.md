# Mini-batch 3-B - PlayerHQ Detail+tabs Del 1 - 5 skjermer

**5 skjermer i denne mini-batchen.** Felles moenster: Header + tabs + tier-gating (Pro/Elite). Spilleren ser planen, oekt-detaljer, tester, maal og TrackMan-data fra eget perspektiv.

**Generer alle 5 skjermer i denne sesjonen.** Foelg arketype-C-felles spec og pass paa lock-overlay paa Free-tier.

---

## Felles arketype-C spec (gjelder alle skjermer her)

(Se 3-A.md for fullstendig arketype-C-spec.)

### Tier-gating-konvensjoner

- **Free-tier:** Hele tab-innhold med 40 % opacity + sentrert lock-card med Lucide Lock + CTA `Oppgrader til Pro ->`
- **Pro-tier:** Standard tilgang. Elite-features synlige med lock-overlay i kort-form.
- **Elite-tier:** Alt unlocked.

### PlayerHQ-spesifikt

- Sidebar er ETT-LAGS (ikke to-lags som CoachHQ): 240px lys nav-kolonne med ikoner
- Avatar i sidebar-toppen viser spillerens profilbilde + tier-pill
- Spilleren ser sin egen plattform -- alt er "deg-perspektiv", ikke "spilleren"
