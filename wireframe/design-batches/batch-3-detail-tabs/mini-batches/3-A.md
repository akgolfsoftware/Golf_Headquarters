# Mini-batch 3-A - CoachHQ Detail+tabs - 5 skjermer

**5 skjermer i denne mini-batchen.** Felles moenster: Header-blokk + tab-strip + tab-innhold (asymmetrisk grid). Coach jobber daglig med disse skjermene -- 360-profil, plan-detalj, plan-edit, talent-pipeline, light spiller-detalj.

**Generer alle 5 skjermer i denne sesjonen.** Foelg arketype-C-felles spec for fellestrekk (header, tab-strip, sticky action-strip, drawer-konvensjoner).

## Felles arketype-C spec (gjelder alle skjermer her)

# Batch 3 -- Detail + tabs (Arketype C)

Disse moenstrene skal vaere konsistente paa tvers av alle detail-skjermer. Vis variasjon i innhold per tab, ikke i layout.

### Layout

```
+------------------------------------------------+
|  Sidebar  |  Breadcrumb (Liste > Navn)         |
|           |  +------------------------------+  |
|           |  |  Header-blokk:               |  |
|           |  |  Avatar (64px) + Navn (H1)   |  |
|           |  |  + Subtittel + 4 stat-pills  |  |
|           |  |  + Primary CTA + Action-meny |  |
|           |  +------------------------------+  |
|           |  Tab-strip (sticky):               |
|           |  [Aktiv]  Tab 2   Tab 3   Tab 4    |
|           |  -------                           |
|           |  +------------------------------+  |
|           |  |  Tab-innhold                 |  |
|           |  |  (asymmetrisk grid, bento)   |  |
|           |  +------------------------------+  |
|           |  Sticky bottom action-strip        |
+------------------------------------------------+
```

### Header-blokk -- UNIKT for arketype C

- **Avatar/ikon:** 64px (sirkel for personer, rounded-lg for ting)
- **H1-tittel:** Geist 32px, eller Instrument Serif italic for hero-effekt
- **Subtittel:** muted 14px (rolle, kategori, dato -- relevant kontekst)
- **Stat-pills (3-4):** smaa chips med JetBrains Mono for tall (HCP, %, kr, dato)
- **Primary CTA:** hoeyre side
- **Action-meny `...`:** sekundaere handlinger (Eksporter, Rediger, Slett)

### Tab-strip

- Horisontal liste, default 4-7 tabs
- **Aktiv tab:** 2px stripe under tab-tekst (primary)
- **Inaktiv:** muted-foreground, ingen stripe
- **Hover:** subtil bg-shift
- **Sticky** ved scroll

### Tab-innhold -- generelle prinsipper

- **Asymmetrisk grid** (12-col, ikke 3x1 uniform)
- **Bento-card-moenster:** 2 store kort + 3-4 smaa kort, evt. en bred chart-stripe
- **States per tab:** loading skeleton, empty (med dempet ikon + CTA), error med retry
- **Tab-bytte:** instant (ingen page-reload), tab-state lagret i URL hash

### Stat-rich-cards

- Stor verdi (Geist 32px, tabular-nums)
- Liten label over (muted 12px)
- Sparkline eller delta-indikator under
- Klikk -> drill-in til relatert tab eller drawer

### States aa designe

| State | Hvor |
|---|---|
| Loading per tab | Skeleton-blokker i samme grid som tab-innhold |
| Empty per tab | Dempet ikon + "Ingen X ennaa. Lag din foerste ->" |
| Error per tab | Per-tab-feilmelding med retry-knapp |
| Tab-bytte-animasjon | Crossfade 150ms |
| Header-collapse paa scroll | Komprimer header til 56px sticky bar |
| Print/PDF-versjon | Skjul sidebar + tab-strip, vis alt innhold |

### Mobil

- Header: avatar + tittel stables, stat-pills i horisontal scroll-rad
- Tab-strip: horisontal scroll, sticky
- Tab-innhold: 1-kolonne, alle cards stables
- Sticky action-strip: bunn-festet

## Spec -- hva skjermen er for

Coachens dypeste view av en spiller. Fra Elever-listen klikker coach inn hit for aa se hele bildet -- pyramide-fokus over tid, SG-utvikling, TrackMan-data, test-resultater, plan-status, turneringer og notater. 40+ klikkbare elementer.

- **Subtittel:** `Kategori A - Elite - WANG Toppidrett - 17 aar`
- **Stat-pills (4):** `HCP +2,4` - `SG +0,8 (12u)` - `Sist trent: i dag` - `Plan: Sommer-toppform`
- **Primary CTA:** `Book ny oekt` (aapner BookSessionModal)
- **Sekundaer:** `Send melding` + `...`-meny

## Tab-strip (7 tabs)

| Tab | Innhold |
|---|---|
| **Pyramide** (default) | Donut med 5-tier fordeling + heatmap |
| **SG** | Strokes Gained-stripe per kategori + sparklines |

| **Tester** | Tabell over tester med personlig rekord (*) |
| **Plan** | Aktiv plan-card med fase-fremdrift |
| **Tournaments** | Kommende + 6 siste resultater |
| **Notater** | Coach-notat-feed (10 siste) |

## Pyramide-tab (default) layout

- **Stort donut-card (8-col):** Pyramide siste 4 uker, klikk-segment aapner drawer
- **Stat-rich 2x2 (4-col):** FYS 18 % / TEK 32 % / SLAG 24 % / SPILL 14 % / TURN 12 %
- **Heatmap-stripe (12-col):** 12 uker x 5 omraader
- **Hvor henger han? (12-col):** 5 stat-cards horisontalt

## Eksempel-data

- Spiller: Markus Roinaas Pedersen, A-kategori Elite
- Pyramide: TEK 32 % - SLAG 24 % - FYS 18 % - SPILL 14 % - TURN 12 %
- Aktiv plan: "Sommer-toppform" -- fase 3 av 5 (Spesifikk), 64 % gjennomfoert
- Coach: Anders Kristiansen

3. Tab-bytte til SG
4. Drawer aapen: PyramidTierDetailDrawer for "Slag (24 %)"
