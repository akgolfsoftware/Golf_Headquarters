# Mini-batch 5-B - Tverrgaaende fullscreen + Live Session 4-modal-pakke

**6 enheter i denne mini-batchen:** 2 tverrgaaende fullscreen-skjermer + 4 Live Session-modaler som henger sammen i en flyt.

**Generer alle 6 i denne sesjonen.** Foelg arketype-E-felles-spec. Live-modalene (pakke 3-6) MAA designes som en sammenhengende flyt -- intro -> active -> between -> summary.

---

## Felles arketype-E spec (gjelder alle skjermer her)

# Batch 5 -- Fullscreen / Live (Arketype E)

### Layout -- fullscreen, ingen sidebar

```
+---------------------------------------------+
|  [Live - Markus R]              [Lukk x]    |  <- topp-bar 56px
+---------------------------------------------+
|                                             |
|         STORT SENTRERT INNHOLD              |
|                                             |
+---------------------------------------------+
|  [Pause]              [Neste rep ->]        |  <- bunn-bar 88px
+---------------------------------------------+
```

**Sidebar er skjult helt.** Topp-bar er minimal. Bunn-bar inneholder primaer handling (alltid 88px hoey for tommel-rekkevidde).

### Bakgrunn

| Modus | Bakgrunn |
|---|---|
| Agent Pipeline Overview | #0A1F18 med svak grid-pattern (8px raster, 4% opacity) |
| Periodiserings-Agent | #0A1F18 |
| LiveIntroModal | #0A1F18 med subtil noise-grain |
| LiveActiveModal | #0A1F18 |
| LiveBetweenModal | #0A1F18 med pulserende grad-bevegelse (breathing-effekt) |
| LiveSummaryModal | #0A1F18 med subtil konfetti-grain (lime-prikker svever oppover) |

### Tap-targets

- **Minimum 44 x 44px** for alle interaktive elementer
- **Primary CTA i bunn-bar:** 88px hoey, full bredde minus 32px padding, rounded-full
- **Sekundaer:** 56px hoey

### Counter-typografi

- Rep-teller: JetBrains Mono 120px, weight 500, tabular-nums
- Klokke / nedtelling: 64px (LiveBetween) / 96px (pust-nedtelling)
- Hero-tid LiveSummary: 64px lime accent
- Subtekst under counter: Geist 14px muted

### "Avslutt" / "Lukk"-knapp -- alltid topp-hoeyre

40 x 40px sirkel, rgba(245,244,238,0.08) bg, lucide X-ikon (24px, #F5F4EE).

## Pakke 1/6: Agent Pipeline Overview (Tverrgaaende)

# AK Golf Platform -- Shared -- Agent Pipeline Overview

## Identitet

- **Produkt:** Tverrgaaende (PlayerHQ + CoachHQ)
- **URL:** /agents/overview
- **Arketype:** E -- Fullscreen / Live (full-system kart)
- **Tier-gating:** Synlig for alle, interaktive node-detaljer krever Pro+
- **HTML-referanse:** wireframe/screen-deck/shared/cross-cutting/agent-pipeline-overview.html

## Spec -- hva skjermen er for

Full-system-kart over **alle** agenter, signaler og skills i hele plattformen. Brukes baade av spillere (forstaaelse) og coacher (debugging og tillit). Viser hvordan data flyter fra TrackMan, GolfBox, Deepgram, soevn-tracker, etc., gjennom skills til agenter.

## Layout -- UNIKT

Bruker arketype-E-felles-spec. **Fullscreen, ingen sidebar.** Bakgrunn #0A1F18 med svak grid-pattern.

### Topp-bar (56px)

- **Venstre:** "Agent Pipeline -- System Overview" Geist 14px
- **Senter:** Tabs: Live aktivitet / Statisk kart (segmentert kontroll)
- **Hoeyre:** Lukk-X 40x40px

### Senter -- node-graf (3 kolonner, full bredde)

- **Signal-noder (8 venstre):** 24px sirkler, lucide-ikoner (TrackMan, GolfBox, Deepgram, Soevn, ScoreCard, Live, Helse, Manual)
- **Skill-noder (5 senter):** 240x80px kort (Tech-analyse, Volum-analyse, Moenster-analyse, etc)
- **Agent-noder (4 hoeyre):** 96x96px sirkler (Drill, Periodisering, Deload, Escalation)
- **Linjer:** 2px stroke #2B4F42, animert dash naar data flyter, glow #D1F843 ved output

### Bunn-bar (88px)

- Venstre: "8 signaler - 5 skills - 4 agenter -- sist oppdatert 14:32" muted
- Hoeyre: Send agent-feedback -> (primary lime ghost)

## OEnsket output

1. Live aktivitet -- flere noder aktive samtidig, linjer pulserer
2. Statisk kart -- alle frosset, hover-tooltip vises
3. Agent-node klikket -- popover aapen
4. Empty (ingen aktivitet)
5. Mobil -- vertikalt stacket, swipe-down for aa bla
6. Moerkt tema er default (kun moerk variant)

## Pakke 2/6: Periodiserings-Agent (4-lags decision-stack)

# AK Golf Platform -- Shared -- Periodiserings-Agent

## Identitet

- **Produkt:** Tverrgaaende (CoachHQ primaer, PlayerHQ read-only)
- **URL:** /agents/periodisering
- **Arketype:** E -- Fullscreen / Live (decision-stack)
- **Tier-gating:** Pro+ ser hele decision-stacken
- **HTML-referanse:** wireframe/screen-deck/shared/cross-cutting/periodiserings-agent.html

## Spec -- hva skjermen er for

Periodiserings-agenten er den mest komplekse av plattformens agenter. Denne skjermen visualiserer **decision-stacken** slik at coach kan se hvorfor agenten foreslo det den foreslo. Fire lag: **L1 Maal** -> **L2 Pyramide** -> **L3 Volum** -> **L4 Periodisering**.

## Layout -- UNIKT

Bruker arketype-E-felles-spec. **Fullscreen, ingen sidebar.** Bakgrunn #0A1F18.

### Topp-bar (56px)

- **Venstre:** "Periodiserings-Agent" + spiller-avatar (Markus R) 24px
- **Senter:** "Anbefaling generert 14:32 - Ikke godkjent ennaa" muted
- **Hoeyre:** Lukk-X 40x40px

### Senter -- 4-lags decision-stack (vertikal)

Hvert lag fullbredde-kort med tydelig avgrensning, koblet med 2px vertikal linje:

- **L1 MAAL** -- "Soerlandsaapent 14. juli -- toppform uke 28"
- **L2 PYRAMIDE-FORDELING** -- "TEK 40% - SLAG 25% - FYS 15% - SPILL 12% - TURN 8%" + horisontal stacked bar 64px
- **L3 VOLUM** -- "14 timer/uke - oppbygning fra 11 t" + bar-chart 8 uker
- **L4 PERIODISERING** -- "Mai: oppbygning - Juni: peak - Uke 28: peak" + tidslinje-graf

Hvert lag har chevron-toggle (collapsed/expanded).

### Bunn-bar (88px)

- Venstre 33%: Avslaa med aarsak (secondary)
- Senter 34%: Detaljer -> (ghost)
- Hoeyre 33%: Aksepter anbefaling (primary lime, 88px)

## OEnsket output

1. Default -- alle 4 lag collapsed
2. L2 expanded -- pyramide-mini-chart synlig
3. Alle expanded -- full scrolled view
4. Allerede godkjent (status-tekst i bunn)
5. Avslaatt-state med aarsak
6. Mobil -- lagene full bredde, swipe-down paa header for collapse
7. Tier-gate (Free) -- L2/L3/L4 blurred
