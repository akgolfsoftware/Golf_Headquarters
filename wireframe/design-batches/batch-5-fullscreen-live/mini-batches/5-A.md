## Felles arketype-E spec (gjelder alle skjermer her)

# Batch 5 -- Fullscreen / Live (Arketype E)

**Antall pakker:** 12 (8 fullscreen-skjermer + 4 Live Session-modaler)

## Hvorfor denne batchen naa

Arketype E er den mest visuelt avvikende moensteret i hele plattformen. Disse skjermene tar over hele viewport, fjerner sidebar og top-bar, og bruker moerk bakgrunn med store tap-targets. Dette er hvor coach og spiller gaar "into the zone".

## Arketype E -- felles spec

### Layout -- fullscreen, ingen sidebar

```
+---------------------------------------------+
|  [Live - Markus R]              [Lukk x]    |  <- topp-bar 56px
+---------------------------------------------+
|                                             |
|         STORT SENTRERT INNHOLD              |
|         (counter, chat, board)              |
|                                             |
+---------------------------------------------+
|  [Pause]              [Neste rep ->]        |  <- bunn-bar 88px (tap-targets)
+---------------------------------------------+
```

**Sidebar er skjult helt.** Topp-bar er minimal (kun kontekst-label venstre + Lukk/Avslutt hoeyre). Bunn-bar inneholder primaer handling (alltid 88px hoey for tommel-rekkevidde).

### Bakgrunn og atmosfaere

| Modus | Bakgrunn | Naar |
|---|---|---|
| Live Session (PlayerHQ) | #0A1F18 (deep-pyramide-green) | Tap-mode, rep-logging |
| Live Tapper (range) | #000000 (helt sort) | Range-modus, full fokus |
| Sesjonsopptak (CoachHQ) | #0A1F18 med subtil noise-grain | Deepgram pipeline kjoerer |
| AI-chat (Coach Agent) | #0F1F1A (litt lysere moerk) | Chat-modus med god lesbarhet |
| Coaching Board (kanban) | #0A1F18 med kolonne-bg #163027 | Board-modus, dra-funksjon |

Tekst er alltid lys (#F5F4EE eller #FAFAF7) paa moerk. Accent-lime (#D1F843) brukes sparsomt for primaer handling og counter-tall.

### Tap-targets og interaksjon

- **Minimum 44 x 44px** for alle interaktive elementer
- **Primary CTA i bunn-bar:** 88px hoey, full bredde minus 32px padding, rounded-full
- **Sekundaer handling:** 56px hoey, ved siden av primaer eller stacked
- **Counter-tap-zone (Live):** hele midten av skjermen er tap-zone som logger rep -- visuelt feedback med kort flash av accent
- **Lange-tap (long-press):** 600ms for aa trigge sekundaer handling

### Counter-typografi

Live-counter alltid sentrert med **JetBrains Mono store tall**:

- **Rep-teller:** 120px font-size, font-weight: 500, tabular-nums
- **Klokke / nedtelling:** 64px, samme stil
- **Subtekst under counter:** Geist 14px muted (#9D9C95)
- **Pyramide-fokus-pill over counter:** Geist 12px uppercase letterspacing 0.08em

### "Avslutt" / "Lukk"-knapp -- alltid topp-hoeyre

Posisjon: top: 16px, right: 16px. Form: 40 x 40px sirkel, rgba(245,244,238,0.08) bg, lucide X-ikon (24px, #F5F4EE).

Klikk -> bekreftelse-popover ("Er du sikker? Sesjonen lagres ikke") med to knapper.

### Minimal UI -- kun essensielle kontroller

Ingen sidebar. Ingen breadcrumbs. Ingen secondary navigation. Ingen filter-bar. Ingen KPI-kort.
