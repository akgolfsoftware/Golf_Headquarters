# Batch 4 -- Wizard / Form (Arketype D)

**Antall pakker:** 18 (12 skjermer + 6 modaler)

## Arketype D -- felles spec

Disse moenstrene skal vaere konsistente paa tvers av alle wizards og forms. Vis variasjon i innhold og felter, ikke i layout.

### Layout (multi-step)

```
+---------------------------------------------+
|  Sidebar  |  Hero (italic editorial title)  |
|           +---------------------------------+
|           |  Steg-indikator (dots/numbers)  |
|           |  o --- o --- O --- O --- O      |
|           +---------------------------------+
|           |  Body (variabel hoeyde)         |
|           |  +---------------------------+  |
|           |  | Felt 1                    |  |
|           |  | [_________________]       |  |
|           |  |                           |  |
|           |  | Felt 2                    |  |
|           |  | [_________________]       |  |
|           |  +---------------------------+  |
|           +---------------------------------+
|           |  Footer (sticky, 72px)          |
|           |  [Avbryt]      [<- Tilbake] [->]|
|           +---------------------------------+
```

### Felles regler

| Element | Regel |
|---|---|
| Steg-indikator | Dots eller numbers -- ALDRI lineaer progressbar |
| Aktiv steg-prikk | accent (lime) for aktiv, primary for fullfoert, muted for ufullfoert |
| Klikkbar steg-prikk | Brukere kan hoppe TILBAKE til fullfoerte steg, ikke fremover |
| "Avbryt"-knapp | Alltid synlig (sekundaer) -- venstre i footer |
| "Tilbake"-knapp | Disabled paa steg 1 |
| "Fortsett ->" / "Neste ->" | Primary CTA (accent-pill) -- lengst til hoeyre |
| Submit-CTA | Bytter tekst: "Send forslag ->", "Bestill naa", "Opprett konto", "Lagre" |

### States aa designe

| State | Naar |
|---|---|
| Idle | Default -- ingen interaksjon |
| Focus per felt | Klikk inn i felt (ring-token) |
| Validating | Async-validering -- spinner i felt |
| Inline error | Roed tekst under felt + ikon |
| Inline success | Kun paa kritiske felt (passord, e-post) |
| Submitting | CTA spinner + footer disabled |
| Success (final) | Confetti / checkmark / redirect-melding |
| Error (submit) | Toast oeverst i form |

### Validering

- Real-time: e-post-format, passord-styrke, kort-nummer, telefonnummer
- On-blur: navn, valgfrie felt, dropdowns
- On-submit: helhets-validering

### Mobile (<=640px)

- Steg-indikator komprimerer til "Steg 2 av 5" tekst + dots
- Footer sticky-bunn med full-bredde knapper
- Avbryt blir tekst-link i topp-hjoernet
