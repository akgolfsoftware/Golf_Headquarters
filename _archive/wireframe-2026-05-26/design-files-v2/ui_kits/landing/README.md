# Landing UI Kit

Marketing landing page. Dark editorial hero (`#0A0A0A`) + lime CTA, then alternating cream/sand sections with feature grid, split content, primary-green CTA strip, dark footer.

## Screens
- `index.html` — full landing flow: nav, hero, three-up features, split visual, primary CTA strip, footer.

## Component vocabulary
- **Hero h1** — Inter Tight 80px bold; the italic word in lime is the brand "moment".
- **Eyebrow tag** — translucent lime pill with leading dot, marks status/section.
- **CTA primary** — lime fill `#D1F843`, ink-on text. CTA secondary is outlined on dark.
- **Feature cards** — standard cards with a soft lime icon-tile (44px, `rgba(209,248,67,.18)`).
- **CTA strip** — full-width primary-green panel inside cream surface; lime button on top.

The hero stats are absolute-positioned bottom-right — replace with hero photography (golf course, desaturated warm tone) when assets land. Style guide spec: `photo + dark overlay 0.65`.
