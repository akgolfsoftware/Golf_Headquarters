# Mini-batch 7-A - CoachHQ kalender / kapasitet / finance / facilities / reports

**5 skjermer i denne mini-batchen.** Felles moenster: spesial-visninger uten ett enkelt moenster. Hver er sin egen IA - foelg per-skjerm-spec.

**Generer alle 5 skjermer i denne sesjonen.**

---

## Felles arketype-G spec (gjelder alle skjermer i batch 7)

Arketype G er paraplyen for "alt annet" - hver skjerm beskriver sitt eget unike layout. Felles regler holder visuell konsistens med batch 1-6:

### Felles regler

- **Hero alltid italic Instrument Serif** med editorial fragment, aldri "Velkommen, ..."
- **Lyst tema default**, moerkt tema som speilet variant
- **8pt-grid** for spacing (8/16/24/32/40/48/64). Aldri p-1, p-3, p-5, p-7, p-9
- **Lucide-ikoner**, 1.75 stroke, alltid currentColor
- **Maks 3 lime-elementer** (#D1F843) synlig per skjerm
- **Norsk bokmaal** med AE/OE/AA, komma som desimal (12,4), mellomrom som tusenseparator (1 600 kr)
- **24-timer**, ISO-dato i datapunkter ("11. mai 2026")
- **Asymmetrisk layout** naar relevant - ikke 3x1 uniform-grid hvis innholdet kan rytmiseres

### Spesial-visninger - kalendere og grids

Pakke 1 (Kalender) og 2 (Kapasitet) bruker tids/heatmap-grid. Felles regler:

- Tidsblokker fra 06:00 til 22:00, 30-min slots
- "Naa"-linje (roed horisontal, 1px) paa dagens dato/klokkeslett
- Pyramide-stripe (4px venstre paa blokk) hvis oekt har FYS/TEK/SLAG/SPILL/TURN-fokus
- Klikk paa blokk -> quick-popover med detaljer + 2 aksjoner
- Klikk paa tom slot -> modal med pre-fylt tid

### Empty / loading / error

- Empty: sentrert ikon + tekst + CTA
- Loading: skeleton som matcher faktisk layout
- Error: per-seksjon-error med retry, aldri full-screen-roed

### Mobil

- Spesial-visninger: tids-grid blir 1-dag-view, kart blir kort-liste, finance-tabeller scroller horisontalt
- Katalog-flater: grid kollapser fra 3- til 1-kolonne, filter blir bottom-sheet
