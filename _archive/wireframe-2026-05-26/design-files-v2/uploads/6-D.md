# Mini-batch 6-D - CoachHQ verktoey + 3 modaler

**7 enheter i denne mini-batchen.** 4 CoachHQ-verktoey (grupper utvidet, e-post-maler, agent-konfig, teknisk plan) + 3 sentrale modaler (avatar-upload, change-password, cancel-subscription).

**Generer alle 7 enheter i denne sesjonen.** Foelg arketype-F-felles-spec for skjermene; modalene foelger modal-konvensjoner.

---

## Felles arketype-F spec

Samme som 6-A/B/C. Modaler har egne moenstre:
- Centrert, max-bredde 480-560px
- Header: tittel + close-X
- Body: form eller wizard
- Footer (action-bar): Avbryt (ghost) | Primary (eller destructive)
- Mobil: bottom-sheet (full bredde, slide-up fra bunn)
- Esc + klikk utenfor lukker (med confirm hvis dirty)

## Pakke 6/7: Modal ChangePasswordModal

# AK Golf Platform -- Modal -- ChangePasswordModal

## Identitet

- **Produkt:** Shared (CoachHQ + PlayerHQ)
- **URL:** Modal - aapnes fra settings-skjermer
- **Arketype:** F (sikkerhets-modal)
- **HTML-referanse:** `wireframe/screen-deck/shared/modals/change-password.html` (lag ny)

## Spec

Krever 3 ting: naavaerende passord (re-auth), nytt passord (med live styrke-validering), bekreftelse. Etter suksess: andre sesjoner logges ut.

## Layout

Modal centrert, max-bredde 480px.

**Header:** "Endre passord" + close-X.

**Body:**

Felt 1 - Naavaerende passord: input (type=password) + "Vis"-toggle (eye-ikon). Helper "Bekreft hvem du er".

Felt 2 - Nytt passord: input + "Vis"-toggle. **Styrke-meter** (4-segment-bar roed/oransje/gul/groenn) + tekst "Svakt/Middels/Bra/Sterkt". **Krav-sjekkliste** (live, hver linje faar checkmark naar oppfylt):
- x Min 12 tegn
- x Minst 1 stor bokstav
- x Minst 1 tall
- x Minst 1 spesialtegn
- x Ikke samme som tidligere passord
- x Ikke et vanlig passord

Felt 3 - Bekreft nytt passord: input. Live-validering "Match" (groenn) eller "Stemmer ikke" (roed).

Toggle "Hold meg innlogget paa denne enheten" (paa default). Tekst "Andre enheter logges ut for sikkerhet".

**Footer:** Avbryt (ghost) | Lagre nytt passord (primary, disabled inntil alle krav + match).

## States

1. Tom (default) - alle krav muted, Lagre disabled
2. Skriver - krav faar checkmarks live, styrke-meter oppdaterer
3. Lagrer - Lagre-knapp viser spinner
4. Suksess - check + "Passord endret", lukker etter 2 sek
5. Feil - inline roed tekst ("Naavaerende passord stemmer ikke" osv)

## OEnsket output

1. State 1 (tom) lyst tema
2. State 2 (skriver, 4/6 krav, styrke "Bra")
3. State 4 (suksess)
4. State 5 (feil naavaerende passord)
5. State 1 moerkt tema
6. Mobil <=640px (full bredde, sticky bunn-action-bar)

## Pakke 7/7: Modal CancelSubscriptionModal

# AK Golf Platform -- Modal -- CancelSubscriptionModal

## Identitet

- **Produkt:** PlayerHQ + CoachHQ
- **URL:** Modal - aapnes fra abonnement-skjermer
- **Arketype:** F (retention-modal)
- **Tier-gating:** Pro/Elite (Free har ikke noe aa kansellere)
- **HTML-referanse:** `wireframe/screen-deck/shared/modals/cancel-subscription.html` (lag ny)

## Spec

Retention-flow med bevisst friksjon. 3 steg (aarsak -> tilbud -> bekreft).

## Layout

Modal centrert, max-bredde 560px. 3-stegs wizard inni (progress-bar oeverst).

**Step 1 av 3 (aarsak):** Tittel "Vi blir lei oss for aa se deg gaa". Sub "Si hvorfor - saa kan vi bli bedre". Radio-gruppe:
- For dyrt
- Bruker det ikke nok
- Mangler funksjoner jeg trenger
- Bytter til konkurrent
- Skader / midlertidig pause
- Annet

Hvis "Annet": textarea aapnes.
Action: Avbryt | Neste ->.

**Step 2 av 3 (personlig tilbud, basert paa aarsak):**

For dyrt: "Vi tilbyr 30% rabatt i 3 maaneder - 209 kr/mnd i stedet for 299 kr". Aksepter rabatt-tilbud (accent CTA) eller "Nei takk, fortsett kansellering ->".

Bruker ikke nok: "Pause i stedet". Pause i 1/3 mnd.

Skader: "Vi pauser gratis saa lenge du trenger. Coach Anders faar varsel". Aksepter gratis pause.

Mangler funksjoner / Bytter: "Send oss en mail om hva som mangler". Send feedback ->.

**Step 3 av 3 (bekreft):** Tittel "Bekreft kansellering". Sammendrag-kort (Plan Pro 299 kr/mnd, Aktiv til 1. juni 2026 - 21 dager). "Du mister tilgang til:" 3-punkts liste. "Du beholder:" data 30 dager + profil + runde-historikk. Toggle "Slett alle data umiddelbart" (av default - destructive). Tekst-bekreftelse-input "Skriv 'KANSELLER' for aa bekrefte". Action: Tilbake | Bekreft kansellering (destructive primary, disabled inntil "KANSELLER").

**Suksess-state:** Stor checkmark + "Kansellert". "Pro gaar ut 1. juni. Vi sender bekreftelse paa mail." Knapp "Tilbake til abonnement ->".

## OEnsket output

1. Step 1 (aarsak-radio) lyst tema
2. Step 2 (rabatt-tilbud "For dyrt") med accent-CTA prominent
3. Step 2 (gratis pause for "Skader") - empati-design
4. Step 3 (bekreft) med farezone-toggle og tekst-bekreftelse
5. Suksess-state etter kansellering
6. Mobil <=640px (modal full bredde, sticky bunn)
