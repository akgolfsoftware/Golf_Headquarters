## Designsystem

Bruk `branding-style-guide.html` + `design-system-v2.md` + `web/_shared.css`.

## Spec — hva siden er for

500-side for server-feil. Skal kommunisere at det er var feil, ikke
brukerens. Lavere selvtillit-tone enn 404, mer apologetisk men ikke
panisk. Inkluder error-ID for support-haandtering.

### 2. Sentrert hero (lys, full viewport-hoyde minus nav + footer)

- 3 alternativer som cards i grid-3:
  - **Last inn paa nytt** (Lucide RefreshCw)
