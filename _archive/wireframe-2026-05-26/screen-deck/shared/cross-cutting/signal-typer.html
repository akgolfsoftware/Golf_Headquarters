Signal-typer

Alle signaler systemet kan generere. Hver type har severity, trigger-betingelse og payload-eksempel.

missing_sessionWARN
Trigger: spiller har ikke logget økt > 7d.
{ player_id, days_since: 9 }
round_loggedINFO
Trigger: ny runde fra GolfBox eller manuell logg.
{ player_id, course, score: 78, sg: +1.2 }
handicap_changeWARN
Trigger: HCP endret > 1.0 siste 30d.
{ player_id, old: 8.4, new: 7.4, delta: -1.0 }
tournament_lt_21dURGENT
Trigger: hovedmål-turnering < 21 dager.
{ player_id, event, days: 19 }
injury_reportedURGENT
Trigger: spiller eller coach melder skade.
{ player_id, type: "ankel", severity: "moderat" }
test_completedINFO
Trigger: NGF/Team Norway-test levert.
{ player_id, test: "bunker", result: "7/10" }
plan_driftWARN
Trigger: pyramide-allokasjon ute av band > 15%.
{ player_id, tier: "TURN", drift: -22 }
booking_createdINFO
Trigger: ny booking opprettet i kalender.
{ booking_id, facility, start, coach }
booking_cancelledWARN
Trigger: booking avbestilt < 24 timer før.
{ booking_id, hours_before: 8 }
invoice_overdueURGENT

{ invoice_id, amount: 2400, days_late: 21 }

{ from, channel: "app", excerpt }

{ plan_id, from: "build", to: "taper" }

Trigger: spiller traff milestone (HCP, streak, runde-mål).
{ player_id, achievement: "100 økter" }
