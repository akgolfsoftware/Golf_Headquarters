## Pakke 1/5: Shared CBAC matrise

# AK Golf Platform -- Shared -- CBAC matrise

## Identitet

- **Produkt:** Shared / cross-cutting (admin-flate)
- **URL:** `/admin/cbac`
- **Arketype:** F (rettighets-matrise)
- **Tier-gating:** Kun super-admin (Anders) ser hele matrisen
- **HTML-referanse:** `wireframe/screen-deck/shared/cross-cutting/cbac-matrise.html`

## Spec

CBAC = Capability-Based Access Control. Matrisen viser hvilke roller (Coach, Hovedcoach, Admin, Spiller, Forelder, Klubb-ansvarlig) har hvilke capabilities. Anders bruker denne naar han skal gi en ny coach-assistent tilgang.

## Layout

**Kontekst-velger (3 segmenterte knapper):** Per rolle (default) | Per capability | Per bruker (override).

**Hovedmatrisen (per rolle):** Tabell med rader = capabilities (gruppert i seksjoner: Plans / Players / Bookings / Billing / Settings / Audit), kolonner = roller.

Eksempel-rader (~14 capabilities synlig):
- plan.read.own | Coach x | Hovedcoach x | Admin x | Spiller x | Forelder - | Klubb -
- plan.read.all | - | x | x | - | - | -
- plan.create | x | x | x | - | - | -
- plan.publish | - | x | x | - | - | -
- plan.delete | - | - | x | - | - | -
- player.invite | x | x | x | - | - | x
- player.delete | - | - | x | - | - | -
- billing.read.own | - | - | x | x | x | -
- billing.write | - | - | x | - | - | -
- settings.cbac.write | - | - | x | - | - | -

Cell-content: x (accent) = granted, - (muted) = ikke granted, x* = override fra default.
Hover viser tooltip "Default: ja" eller "Override: gitt 12. mar av Anders K".

**Filter-bar:** Soek capability, Chip Seksjon, Chip Vis kun granted, Sort Alfabetisk / Etter risiko.

**Aksjoner:** Klikk cell -> toggle (med confirm hvis destructive). "Eksporter matrise (CSV)", "+ Ny rolle".

**Audit-link:** "Se siste 100 endringer i CBAC ->".

## OEnsket output

1. Lyst tema, full matrise med 14 capabilities synlig
2. Moerkt tema
3. Cell-hover med tooltip
4. Filter aktivt: kun granted
5. Per-bruker-modus aktiv
6. Mobil <=640px (matrise blir vertikal liste)

## Pakke 4/5: Shared Design-tokens (intern viewer)

# AK Golf Platform -- Shared -- Design tokens

## Identitet

- **Produkt:** Shared / cross-cutting (intern utvikler-flate)
- **URL:** `/admin/design/tokens`
- **Arketype:** F (read-only viewer + copy)
- **Tier-gating:** Admin + utvikler
- **HTML-referanse:** `wireframe/screen-deck/shared/cross-cutting/design-tokens.html`

## Spec

Design tokens-viewer viser hele design-systemet i live-format. Read-only fra UI (tokens endres via PR til globals.css), men har copy-knapper.

## Layout

**Tema-bytter (sticky oeverst):** Lyst (default) / Moerkt / Foelg system - endrer hele skjermens preview live.

**Tab-bar (6 seksjoner):** Farger | Typografi | Spacing | Radius | Shadows | Motion.

### Tab Farger (default)

Grid 3-kolonne med fargekort. Hvert kort:
- Stort fargefelt (96px hoeyde)
- Token-navn (Geist Mono): --background
- Tailwind-utility: bg-background
- HSL-trippel: 60 11% 97%
- HEX: #FAFAF7
- Copy-knapp
- Brukt i: "Layout, Card-bg, Sidebar"

Grupper: Surface, Brand, State, Form, Pyramide.

### Tab Typografi

3 store kort: Geist (sans, default UI - 6 weights), Geist Mono ("1 600 kr - +2,4 HCP - 14:32"), Instrument Serif (italic *"Onsdag morgen. 38 spillere venter."*).

Type-skala-tabell: text-xs (12/16) Captions, text-sm (14/20) Body, ..., text-5xl (48/1) Hero.

### Tab Spacing

8pt-grid visualisert med boxer. Don't-list (gjennomstreket): p-1, p-3, p-5, p-7, p-9.

### Tab Radius

5 boxer: rounded-sm (8px badges), rounded-md (12px inputs/knapper), rounded-lg (16px cards default --radius), rounded-xl (12px), rounded-2xl (16px hero), rounded-full (pills).

### Tab Shadows + Motion

Shadows: ingen (vi bruker borders). Eneste unntak: shadow-sm paa popover.
Motion: --motion-fast (120ms ease-out hover), --motion-default (200ms), --motion-slow (320ms modal).

## OEnsket output

1. Lyst tema, tab Farger
2. Moerkt tema (samme - viser at fargene endrer seg live)
3. Tab Typografi med 3 fonter
4. Tab Spacing med 8pt-grid og don't-list
5. Hover-state paa fargekort (lift + copy-tooltip)
6. Mobil <=640px (tabs scroll, fargekort 2 kolonner)
