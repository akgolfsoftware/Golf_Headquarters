# Mini-batch 3-D - Modaler (8 stk drill-ins fra detail-skjermer)

**8 modaler i denne mini-batchen.** Alle aapner fra detail-skjermer i batch-3 (eller list-skjermer i batch-2). Konvensjon: hvit/moerk modal-bg paa overlay-backdrop, header med ikon + tittel + lukk-X, body med form/data, sticky footer med 1-3 knapper.

**Generer alle 8 modaler i denne sesjonen.**

---

## Felles modal-konvensjon

### Layout-typer

| Type | Bruk | Bredde |
|---|---|---|
| Sentrert modal | Skjema, confirm, share | 480-720px |
| Drawer hoeyrejustert | Lange tekst-traader, edit | 480-640px |
| Bottom-sheet (mobil) | Alle modaler paa <=640px | full bredde, max 90vh |

### Header
- Lucide-ikon 20px (ofte med accent-bg) + tittel (Geist 18-20px) + sub (muted 13px) + `X`-lukk hoeyre

### Body
- Padding 24px
- Maks 3 vertikale seksjoner foer scroll
- Form-fields stables vertikalt (label over input, ikke side-ved-side)

### Footer (sticky)
- Border-top
- 1-3 knapper, primary CTA hoeyre
- Ghost (Avbryt/Lukk) venstre
- Min hoeyde 64px

### Backdrop
- bg-black/40
- Klikk = lukk (med dirty-form-warning hvis ulagrede endringer)

### Animation
- Fade-in 200ms backdrop
- Slide-up + fade modal 250ms ease-out

### States
- Default, loading (skeleton), error (banner topp), success-flash (modal-flash + auto-close 1,5s)
