# Mini-batch 4-B - Wizards i app

**6 skjermer i denne mini-batchen.** Felles moenster: Wizard / Form (Arketype D) inni app-kontekst -- med sidebar synlig (CoachHQ to-lags eller PlayerHQ en-lags).

**Generer alle 6 skjermer i denne sesjonen.** Foelg arketype-D-felles spec for fellestrekk (steg-indikator, validering, footer-knapper).

---

## Felles arketype-D spec (samme som 4-A)

Gjelder alle skjermer her. Multi-step bruker dots eller numbers, IKKE lineaer progressbar. Aktiv steg accent, fullfoert primary, ufullfoert muted. Klikkbar tilbake til fullfoerte steg, ikke fremover. "Avbryt" alltid synlig (sekundaer venstre). "Tilbake" disabled paa steg 1. Primary CTA hoeyre, bytter tekst per steg. Inline validering per felt. Submit-states: idle / validating / submitting / success / error.

### Sidebar i denne mini-batchen

**CoachHQ-skjermer (pakke 1, 2, 6):** TO-LAGS sidebar -- smal moerk rail (56px, #061210) + lys nav-kolonne (200px, #FAFAF7). Active item i nav: rgba(209,248,67,0.30) bg + #0A1F18 tekst.

**PlayerHQ-skjermer (pakke 3, 4, 5):** EN-LAGS sidebar, lys (200px, #FAFAF7). Active item samme accent-stil.

## Pakke 1/6: CoachHQ Plan-bygger

# AK Golf Platform -- CoachHQ -- Plan-bygger

## Identitet

- **URL:** `/admin/plans/builder`
- **Arketype:** D -- 6-step plan-wizard
- **HTML-referanse:** `wireframe/screen-deck/coachhq/plan-builder.html`

## Spec

Anders' kjerneverktoey -- bygger 4-12-ukers periodiserte planer. Alternativt mal eller AI-generering. Output: plan med periode, pyramide-fokus, ukentlig oekt-skjema og milestones.

## Layout

CoachHQ-sidebar venstre. Hoved-content sentrert. Steg-indikator (numbers): "(1) Spiller -- (2) Periode -- (3) Pyramide -- (4) Mal -- (5) OEkter -- (6) Bekreft" sticky 56px topp.

### Steg 1 -- Velg spiller
Soekefelt + spillerliste med avatar/HCP/sist-trent. Klikk -> velger + viser preview-card hoeyre.

### Steg 2 -- Periode
Dato-fra + dato-til. Forslags-chips: "4 / 8 / 12 uker / Hele sesongen". Treningsdager-velger. Antall oekter per uke.

### Steg 3 -- Pyramide-fokus
5 slidere (FYS / TEK / SLAG / SPILL / TURN) summert til 100%. Spiller-historikk-indikator under hver. "AI-forslag basert paa spiller ->"-knapp.

### Steg 4 -- Mal eller blank
3 store kort: "Bruk mal" / "AI-generer" (lime accent stripe) / "Start blank".

### Steg 5 -- OEkter (drag-redigerbar)
Uke-fane (8 stk). Per uke: liste med drag-handle, dag/tid, tittel, pyramide-stripe, varighet. "+ Legg til oekt"-link. Milestone-marker (gull-pill) mellom uker.

### Steg 6 -- Bekreft og send
Sammendrag-card. Sjekkbokser: "Send til Markus" / "Notifiser foresatt" / "Synk Google Kalender". CTA bytter til "Send forslag ->".

## OEnsket output

1. Steg 1 lyst tema (Markus valgt, preview synlig)
2. Steg 2 lyst tema (8 uker, 32 oekter)
3. Steg 3 lyst tema (slidere 100% V)
4. Steg 3 lyst tema, AI-forslag-loading
5. Steg 4 lyst tema (3 kort, AI-generer hover)
6. Steg 5 lyst tema (uke 19, en oekt under drag)
7. Steg 6 lyst tema (sammendrag + sjekkbokser)
8. Submit success ("Planen er sendt til Markus")
9. Moerkt tema (steg 5)
10. Mobil <=640px (steg 3)

## Pakke 3/6: PlayerHQ Ny oekt-wizard

# AK Golf Platform -- PlayerHQ -- Ny oekt-wizard

## Identitet

- **URL:** `/min/okt/ny`
- **Arketype:** D -- 6-step
- **HTML-referanse:** `wireframe/screen-deck/playerhq/ny-okt-wizard.html`

## Spec

Markus legger til egendefinert oekt utenfor coach-planen. 6 steg: maal -> lengde -> fasilitet -> tid -> oevelser -> bekreft. Hvis fasilitet krever booking -> aapner BookingConfirmationModal etter steg 6.

## Layout

PlayerHQ-sidebar. Wizard sentrert max-width 720px. Steg-indikator (dots, ikke numbers) sticky 56px topp.

### Steg 1 -- Hva skal du trene?
3 kategori-kort: Fokusert / Komplett oekt / Lek og laer. Subtype-velger etter valg.

### Steg 2 -- Lengde og intensitet
Varighet segmentert (30/60/90/120 min). Intensitet-slider 1-5. Estimat-kort.

### Steg 3 -- Velg fasilitet
Toggle "Mine vanlige" / "Soek alle". Liste med fasiliteter + ledig-tid-strip. Ev. "Hjemme/Egen tid".

### Steg 4 -- Tid
Dato + klokkeslett (15-min interval). Ledig/opptatt vises live.

### Steg 5 -- OEvelser
Auto-foreslaatt fra Coach-plan (4 oevelser). Legg til fra bibliotek. Drag-handle. Sum med varighet-warning.

### Steg 6 -- Bekreft og start
Sammendrag-card. Sjekkbokser: "Del med coach Anders K." / "Paaminnelse 30 min foer" / "Logg automatisk". To CTA-er: "Lagre som planlagt" + "Start naa ->" (kun synlig hvis tid = naa).

## OEnsket output

1. Steg 1 lyst tema (3 kategori-kort)
2. Steg 2 lyst tema (60 min, intensitet 3)
3. Steg 3 lyst tema (Mine vanlige, 3 fasiliteter)
4. Steg 4 lyst tema, opptatt-warning + 3 forslag
5. Steg 5 lyst tema (5 oevelser + sum)
6. Steg 6 lyst tema (sammendrag + CTA-er)
7. Tier-gating Free-blokk
8. Moerkt tema (steg 3)

## Pakke 6/6: CoachHQ Import-assistent

# AK Golf Platform -- CoachHQ -- Import-assistent

## Identitet

- **URL:** `/admin/import`
- **Arketype:** D -- 3-step import-wizard
- **HTML-referanse:** `wireframe/screen-deck/shared/cross-cutting/import-assistent.html`

## Spec

Importerer data fra GolfBox / CSV / Trackman / Garmin Golf / Excel. 3 steg: kilde -> last opp/koble -> match og bekreft.

## Layout

CoachHQ-sidebar. Wizard sentrert max-width 960px (bredere for preview-tabell). Steg-indikator (numbers): "(1) Velg kilde -- (2) Koble til / Last opp -- (3) Match og bekreft".

### Steg 1 -- Velg kilde
Grid 2x3 med 6 kilde-kort: GolfBox (anbefalt) / CSV / Trackman / Garmin / Excel / Annet (manuell). Logo + beskrivelse per kort.

### Steg 2 -- Koble til / Last opp
**OAuth-flyt** (GolfBox/Garmin): "Koble til ..."-knapp -> ny fane -> tilbake med V.
**Fil-upload** (CSV/Excel/Trackman): drag-drop omraade 300px hoeyde med Cloud Upload-ikon. Eksempel-fil-link.
**Manuell:** stor tekstboks for inn-liming.

### Steg 3 -- Match og bekreft
Splitt-view:
- Venstre: kolonne-mapping-tabell (vaart felt <-> kilde-kolonne dropdown). "Auto-detekt"-knapp. Status per rad: V matchet / ! gjettet / X manuell krevd.
- Hoeyre: live preview 5 foerste rader.

Under: konflikter-haandtering (radio). Validering oppsummert. CTA "Importer {N} rader ->".

## OEnsket output

1. Steg 1 lyst tema (6 kilde-kort, GolfBox anbefalt)
2. Steg 2 lyst tema, drag-drop CSV (drag-over state)
3. Steg 2 lyst tema, OAuth-koblet GolfBox (V + klubb-velger)
4. Steg 3 lyst tema (mapping + preview, 3 konflikter)
5. Steg 3 lyst tema, mapping-dropdown aapen
6. Importer loading (progress 47%)
7. Importer success ("47 spillere importert")
8. Moerkt tema (steg 3)
9. Mobil <=640px (steg 3 stacked)
