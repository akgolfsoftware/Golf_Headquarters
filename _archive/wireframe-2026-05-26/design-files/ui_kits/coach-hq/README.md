# CoachHQ UI Kit

Coach-facing dashboard. Dark icon rail (`#061210`) + light cream secondary nav + white content. Professional, structured.

## Screens
- `index.html` — coach dashboard with KPI row, players-needing-followup table, today's schedule sidecard.

## Component vocabulary
- **Dark rail** — 60px wide, holds 5 icon-only nav slots. Active uses `rgba(255,255,255,.10)` bg + lime icon. The brand mark sits at top.
- **Light secondary nav** — 220px, lists named sections. Active uses `rgba(209,248,67,.30)`.
- **Data table** — borderless rows with `--line-soft` hairlines, JetBrains Mono numerics, status pills in the rightmost column.
- **Schedule card** — chronological list of session blocks with mono timestamps.
