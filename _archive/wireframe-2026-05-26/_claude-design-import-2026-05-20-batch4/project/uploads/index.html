Anbefaling: Option 1 — Linear Steps

Trygg, forutsigbar, matcher portal-DNA (booking-wizard og planlegger bruker samme mønster). Brukere 30–60 år forventer et wizard-mønster ved "Velkommen". Option 3 (Goal First) er sterkt alternativ — mål-først gir sterkere motivasjon, men Linear Steps er mindre risikabelt for en første portal-release.

Strategisk flow: Start med Option 1 som MVP-onboarding. Vurder Option 3 som A/B-test etter 3 måneder for å se om mål-først øker 7-dagers retention. Option 5 (Gamified) reserveres for Junior Academy-tier (under 25 år). Option 2 (Chat) kan tilbys som "Snakk med AK-coach i stedet"-knapp for de som vil ha guidet onboarding.
