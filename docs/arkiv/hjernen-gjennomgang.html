1. TrackMan-opplasting & integrasjon med teknisk plan
P1.0–P10.0 fullsving

Nåværende tilstand (verifisert)

  • • CSV: full loop (parse → match club → update goals)
  • • HTML: kun raw session (manglende shots + match)
  • • Match: kun club-normalisering (begrensede aliases)
  • • Goals: kun 5 metrics (smash, carry, speed, side)
  • • Ingen consecutive / streak / "10 godkjente i rad"
  • • Ingen per-kølle bank
  • • Ingen AI (PDF/bilde)
  • • Begrenset filtrering

Måltilstand (anbefalt)

  • ✓ AI-opplasting (PDF/CSV/bilde fra skjerm)
  • ✓ Full parameter-støtte + filtrering
  • ✓ Per-kølle TM-bank (godkjente referanser)
  • ✓ Parameter-mål i teknisk plan (consecutive)
  • ✓ Range/matte-repetisjon modus + live validering
  • ✓ Toveis: Plan → Upload → Bank → Plan
  • ✓ Workbench-redigering for spiller + coach

4. Booking-systemet — vei til optimalt

Nå: Acuity midlertidig + intern (portal + admin + marketing). Mangler dyp kobling til teknisk plan og TM.

  • • Fase 1: Task i teknisk plan → «Book anbefalt slot» (prefill service/facility/reps)
  • • Fase 2: TM-booking → auto-hint import + valider mot plan-mål
  • • Fase 3: Credit fra abonnement + compliance tracking
  • • Optimalt: Smart availability fra plan + coach load + recurring matte-pakker
