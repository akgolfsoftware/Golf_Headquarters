## AgencyOS

```
Bygg AgencyOS — coachens app i AK Golf HQ — etter den opplastede briefen
(agencyos-komplett-brief.md). Den er fasiten for design, struktur og hvilken
motor som finnes i koden.

1. Bruk «Motoren»-delen av briefen + PLATFORM.md som fasit for data, modeller og
   server-actions. Gjenbruk eksisterende kode — bygg ALDRI datalaget på nytt.
2. Slank struktur: 6 sidebar-seksjoner (Oversikt/Cockpit · Stall · Planlegge ·
   Gjennomføre · Innsikt · Admin). Mørkt tema, desktop-først.
3. Spiller-Workbench er master: klikk på spiller HVOR SOM HELST → rett inn i
   spillerens Workbench. Ingen profil-mellomside.
4. Slå sammen alle duplikater. Bruk navnet «AgencyOS» — aldri «CoachHQ».
5. Bygg KUN nytt: fokus pin + 3 AI-forslag (Cockpit), fellesmelding-flyt, global
   spiller↔gruppe-veksler. Resten er re-kobling.

Regler: High fidelity. Branch design/agencyos-lean, ALDRI main, IKKE merge.

Mangler du noe → «Åpne spørsmål», ikke dikt. Legg fram KORT plan før du bygger.
```

## PlayerHQ

```
Bygg PlayerHQ — spillerens app i AK Golf HQ — etter den opplastede briefen
(playerhq-komplett-brief.md). Fasit for design, struktur og motor.

1. Bruk «Motoren»-delen + PLATFORM.md som fasit for data/actions. Gjenbruk
   eksisterende kode — bygg ALDRI datalaget på nytt.
2. Slank struktur: 5 faner — Hjem · Planlegge · Gjennomføre · Analysere · Meg.
   Lyst tema, mobil-først (430px) + desktop.
3. Coach er IKKE en femte fane — skuff fra høyre. Femte fane er «Meg».
4. Workbench er master: «Planlegge» = ETT trykkpunkt → Workbench (alle
   planleggings-moduser inni). Gjennomføre + Analysere = én flate med faner hver.
5. Bygg KUN nytt: Turnering-detalj, og la trykk på en økt-rad åpne SELVE økten.

Regler: High fidelity. Branch design/playerhq-lean, ALDRI main, IKKE merge.
Øyvind Rohjan / Anders Kristiansen. Norsk bokmål. Kun Lucide. Ingen emoji.
Abonnement: gratis / 300 kr — ingen tier-nivåer, ELITE finnes ikke.
```
