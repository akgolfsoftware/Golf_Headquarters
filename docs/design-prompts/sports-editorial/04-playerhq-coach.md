   severity dots, observasjon, coach-notat
5. **Workshop / Coach-planer** (Arketype F) — neste 4 uker som trinn 01-04
6. **Atlas / Coach-videoer + Caddie-callout + Notater** (Arketype H) —
   videogalleri-grid + Caddie-callout + siste 5 notater som notatbok
