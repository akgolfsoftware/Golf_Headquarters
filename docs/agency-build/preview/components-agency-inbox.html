Filter-pills: Alle (count = ulest-totalen) + 4 type-pills med dot i kategori-farge (lime / blå / grå / okker) og ulest-count per type. Aktiv pill: forest fill + lime tekst. Type-counts er talt fra .unread-rader med matching type-tag, ikke fra hele inboksen — derfor matcher 2 + 1 + 1 + 0 totalen (4).

Rader: 7-kolonne grid — checkbox · type-ikon · avatar · emne+preview · klokke · attachment-pip · severity-dot. Ulest = lime 3 px venstre-marker + lime-tint bg + foreground preview-tekst.

Severity-dot: rød (hi) ved tidsfrist eller bak plan · okker (md) ved godkjenning som venter · blå (lo) info · grønn (ok) bekreftet.

Batch-bar: vises kun når ≥ 1 valgt. Lime-tint bg. "Godkjenn alle" som primary, "Avvis" og "Marker lest" sekundær. "Tøm valg" høyre.

Expand: klikk rad åpner inline detail under (ikke modal). Dashet rail venstre kobler rad til detail. Quote-blokk med lime border + kontekst-strip "VAR PLANLAGT → FORESLÅTT" for plan-endringer. Handlingsrad: Godkjenn (pri) · Foreslå endring · Avvis (danger) · ghost-link Svar/Profil.
