Søk er navigasjon, palette er det viktigste verktøyet. Power-brukere skal sjelden bruke menyer — alt går gjennom ⌘K. Designet er optimalisert for å være rask og lesbart, ikke å imponere.

Preview-pane er ikke en hover-tooltip. Den oppdateres med tastatur-navigering, viser samme rik-data som spiller-panelet, og gjør at coachen kan finne riktig spiller uten å åpne den.

Komplementer co-agenten i B3. Denne palette finner og åpner ting som finnes. B3-palette tolker naturlig språk og foreslår nye handlinger. Begge kjører i samme overlay, samme tastatur, samme mic — bare ulike søke-modi.

Loggføres alltid. Hver palette-bruk logges i audit (B3 sek. 4) som "menneske · ⌘K"-rad — så vi kan måle hva folk faktisk leter etter, og forbedre fuzzy-rangering.
