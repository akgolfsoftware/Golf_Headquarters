Matrise: spillere som rader, tester som kolonner. Hver test har akse-fargedot + navn + enhet + mål-pille. Mål kan være "høyere bedre" eller "lavere bedre" — fargen er semantisk relativ til mål, ikke absolutt tall.

Celler: 64 px høyde · siste måleverdi + delta vs forrige · dato (relativ). Hover gir 4 % scale-up. Klikk åpner detalj.

Fargekoder: OVER MÅL (forest-tint), NÆR ±5 % (warning okker), UNDER MÅL (destructive rød), IKKE TESTET (skrå-stripe på cream-bg). Rød 6 px dot top-right = overdue. Lime ring + pulsende dot = test pågår nå.

Tildel test-knapp: per spiller-rad (siste kolonne). Badge med antall tester som mangler eller er overdue (rød tall). Karl Ludvig (test pågår) viser "Komplett" i stedet for tildel-knappen.

Filter: gruppe-pills i toolbaren (Alle / WANG / GFGK / AKA). Eksport CSV + test-mal + ny test til høyre.
