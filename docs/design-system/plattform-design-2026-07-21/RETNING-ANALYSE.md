# PlayerHQ Analyse — låst retning

**Valgt:** B · Form + nedbrytning  
**Dato:** 2026-07-22  
**Valgt av:** Anders  
**Matcher:** Hjem B + Plan B

## Følelsen (5 setninger)

1. Analyse viser **total form først** (tall + trend), deretter **fire områder**.
2. **Svakeste område** er tydelig uthevet (tykkere rad / mer vekt).
3. **Resept** ligger rett under gapet — konkret økt, ikke «utforsk mer data».
4. Én grønn CTA: **Planlegg dette** → Workbench (1 trykk).
5. Samme språk som Hjem/Plan B: oversikt + handling, klarspråk (nærspill, ikke ARG).

## Ikke denne retningen

- A (bare gapet) — for lite helhet
- C (fortelling først) — kan brukes som *tekst i* B, ikke hovedlayout
