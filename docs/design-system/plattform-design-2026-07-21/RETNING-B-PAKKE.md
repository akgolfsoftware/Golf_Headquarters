## Felles følelse (hele appen skal arve dette)

1. **Oversikt før handling** — tall/status synlig, ikke tom «bare start».
2. **Én grønn hovedknapp** per skjerm — resten er sekundært.
3. **5 sekunder:** du skjønner standpunkt og neste steg.
4. **Klarspråk** — nærspill, innspill, tee-slag; «?» på fagord.
5. **Bro mellom skjermene:** Analyse → Planlegg → Plan/Workbench → Start (Hjem).

## Ikke B

- A = ekstremt minimalt (kun vedtak)
- C = coach-tekst/fortelling først

Deler av C (kort coach-melding) kan legges *inn i* B senere — ikke erstatte layouten.
