Flaggskip-flyter redesignet for færrest trykk, 5-sekunders forståelse og én hovedhandling. Alle funksjoner beholdes — dybde kommer etter neste steg, ikke før.

1. Hva nå?Én hero + én CTA. Resten er støy til det er valgt.
2. 1–2 trykkFra hjem til plan / start økt. Aldri «meny av 6 like kort».
3. KlarspråkNærspill, ikke ARG. Tall med enhet. «?» på fagord.
4. Tom = vei videreAldri blindvei. Alltid ærlig tom + én knapp.
