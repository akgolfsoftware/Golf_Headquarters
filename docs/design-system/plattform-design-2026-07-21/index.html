Regler som gjelder overalt

PlayerHQ (spiller)

  • Alltid lyst (krem #F2F1EA)
  • Mobil først (390)
  • 5 knapper nederst: Hjem · Plan · Gjør · Analyse · Meg
  • Én grønn hovedknapp per skjerm
  • Tall alltid med enhet

AgencyOS (coach)

  • Mørk som standard (kan bytte til lys)
  • Desktop først (1680)
  • Grønn stripe til venstre (ikoner)
  • Lime = CTA / valgt — bare én jobb
  • Mobil = rask triage, ikke full coach-arbeid

De 8 familiene (kjernen)

Alle 361 skjermer tilhører én av disse. Godkjenn familiene først — da er resten «fyll inn innhold».

1 · Shell

Navigasjon og «hvor er jeg»

Bunn-nav spiller · ikon-rail coach

2 · Hub

Hva nå? på 5 sekunder

Én hero + CTA (spiller) · grid (coach)

3 · Liste

Skann, filtrer, åpne

Kort på mobil · tabell på desktop

4 · Detalj

Én enhet i dybden

Stack mobil · split desktop

5 · Wizard

Flerstegs handling

Steg-stripe + Neste/Fullfør

6 · Live

Logg uten støy

Fullskjerm · store knapper

7 · Analyse

Tilstand 5s · årsak 30s

Faner · bro til Workbench

8 · Kalender

Tid + handling

Uke-stripe · coach ukegrid

Anbefalt rekkefølge når vi bygger i kode

  1. Workbench + Hjem + Analysere-bro (allerede delvis i nattens PR)
  2. Live-økt (brief → aktiv → oppsummering)
  3. AgencyOS stall + spiller 360
  4. Bookinger + kalender
  5. Resten per familie (samme mal, nytt innhold)
