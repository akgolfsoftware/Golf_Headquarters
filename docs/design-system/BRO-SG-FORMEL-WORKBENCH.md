# SG → AK-formel → Workbench (designkontrakt)

## Jobben
På 5 sekunder se hvor slag tapes, og på ett trykk åpne Workbench for å planlegge resepten.

## Flyt
1. SG-total + 4 kategorier (OTT/APP/ARG=nærspill/PUTT)
2. Svakest kategori uthevet
3. Diagnose: symptom · grunnlag · resept (pyramide-chip + tekst)
4. Primær-CTA: «Planlegg dette i Workbench» → `/portal/planlegge/workbench?zoom=uke`

## Mapping (kode)
- OTT/APP/ARG → pyramide **SLAG**
- PUTT → **TEK** (default i optimal-session; coach kan overstyre)
- `nesteFokus.handlingHref` + `diagnose.resept` fra `load-min-golf.ts`

## TrackMan
- Side i m V/H · bias ≠ spredning · M/PR-badge

## Forbud
Fabrikkert SG · «kort spill» · CTA uten landingsmål
