# Marketing — UI Kit

`akgolf.no` — offentlig landingsside. Editorial sport-analytics i ren form. Foto-tung hero, datakomposisjon under, manifest-aktig case-section.

## Seksjoner

1. **Hero** — full-bleed foto (fairway, golden hour), italic-eyebrow + display med signaturen `<em>` i lime.
2. **Manifest / what we do** — tre kolonner med eyebrows i mono caps, kort tekst, en stor KPI per kolonne.
3. **Stats showcase** — DataGolf-stil rad med tall: 4–5 enorme KPI-er på dark forest bakgrunn.
4. **Coaches** — to-tre kort med foto + navn + tittel + spesialitet.
5. **Plan-promo** — featured-card med pris og inkludert-liste.
6. **Footer** — minimal, mono links, logo til venstre.

## Komponenter

- `Nav` — sticky topbar, transparent over hero, solid under
- `Hero` — fullbleed foto + gradient + display
- `ManifestCol` — eyebrow + KPI + body
- `BigStatRow` — dark-bg horisontal KPI-rad
- `CoachCard` — foto-kort, navn + tittel
- `PlanCard` — pris + features
- `Footer`
