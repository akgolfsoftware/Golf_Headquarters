# FakturaRad

Fakturarad for Forelder og AgencyOS finans. Beløp høyrejustert i mono.

## Bruk
```jsx
<FakturaRad title="Pro-abonnement juni" subtitle="Øyvind Rohjan" amount="299" status="paid" />
<FakturaRad title="Treningsavgift" amount="1 200" status="overdue" onClick={() => openInvoice(id)} />
```
