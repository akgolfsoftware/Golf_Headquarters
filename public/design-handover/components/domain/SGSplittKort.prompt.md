# SGSplittKort

SG-splitt med en rad per akse (OTT/APP/ARG/PUTT): søylefyll + verdi + delta.

## Bruk
```jsx
<SGSplittKort
  data={{
    ott:  { value: 0.8,  delta: 0.2  },
    app:  { value: 0.3,  delta: -0.1 },
    arg:  { value: -0.2, delta: 0.0  },
    putt: { value: 0.6,  delta: 0.3  },
  }}
/>
```
