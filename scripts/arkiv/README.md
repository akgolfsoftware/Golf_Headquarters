# Arkiverte engangs-scripts

Kjørte engangs-migrasjoner/generatorer — beholdt for sporbarhet (hva som ble
kjørt mot prod og når), skal ALDRI kjøres på nytt uten ny vurdering.
Flyttet hit under R1-oppryddingen 2026-07-12.

NB: Relative import-stier er IKKE justert etter flyttingen (../src peker feil
herfra) — bevisst: mappen er ekskludert fra tsconfig og skal ikke kjøres.
Må et script gjenbrukes: kopier det ut og fiks stiene der.
